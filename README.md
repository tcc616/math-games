# math-games

這是一個數學遊戲總專案，用來收納各式各樣跟數學有關的教學遊戲。

## 專案定位

- 主專案：數學遊戲工作桌
- 子專案：每個遊戲作為此專案底下的分支、子資料夾或後續模組管理
- 對象：數學課堂、練習活動、互動教學工具

## 建議結構

```text
games/
  <game-name>/
shared/
docs/
```

## 遊戲清單

| 遊戲 | 路徑 | 狀態 |
|------|------|------|
| 九九乘法遊戲 | `games/九九乘法遊戲/index.html` | 可直接用瀏覽器開啟 |

## 工作流程

- 開始工作時說「開工」
- 建立新遊戲時，先確認遊戲名稱、學習目標、玩法與資料需求
- 結束工作時說「收工」

## 安全原則

- 不提交 `.env`、API key、token、密碼或管理員憑證
- 學生資料只使用座號與班級代號，不存真名
- Firebase 尚未啟用；等需要資料庫的遊戲再設定
